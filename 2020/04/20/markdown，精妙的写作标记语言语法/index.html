Markdown 学习使用笔记

markdown是一种极简的标记语言，通过将常用到的排版方式用特定的格式表示，免去在写文案时去点击排版格式图标，大大提高写作效率，熟练之后可以忽视掉写作时候烦人的排版工作，从而让写作时更加专心于写作本身。

markdown的标记语言比较简单，是类似于HTML语言的标记语言，首先我们来看
